  • 智能写作

    智能写作是百度公司在人民日报人民号创作平台推出的“创作大脑”，充分利用百度在内容理解、语言生成、知识推理、机器学习等方面的技术积累，为用户提供内容纠错、发文助手、自动封面三项创作服务。

  • 智媒引擎

    智媒引擎通过多种数据挖掘算法、语义分析算法，梳理新闻分类、提取新闻要素、统计媒体报道，帮助用户快速掌握新闻事件全貌、发现新闻热点、滤清事物人物关系、深挖新闻线索，推荐新闻素材。

  • 语音转写

    讯飞听见APP是一款语音转文字的手机应用，依托科大讯飞基于世界领先水平的智能语音及语言技术提供高效的语音转文字服务，识别率高达95%＋，1小时音频最快5分钟出稿！

  • 数据魔方

    基于国内先进的数据采集技术、大数据分析技术，汇集海量数据，进行人工智能分析。帮助媒体以最低的成本，对数据进行深度挖掘。同时促进媒体的采编发融合，让媒体变得更加智慧。

  • 视频搜索

    快手视频搜索引擎利用国际领先的人工智能技术，实现对视频、音频和图像的多媒体理解，可根据关键字、发布时间和发布地点精准搜索快手平台内海量视频资源，为媒体提供源源不断的内容和选题支持。
